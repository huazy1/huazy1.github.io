Seção 2 — Ranking de Tecnologias

  1. HTML

    Linguagem de marcação utilizada para estruturar páginas web e organizar conteúdos.

  2. CSS

    Responsável pela estilização visual das páginas, incluindo cores, layouts e animações.

  3. JavaScript

    Linguagem de programação usada para adicionar interatividade e dinamismo às aplicações web.

  4. React

    Biblioteca JavaScript amplamente utilizada para criação de interfaces modernas e componentes reutilizáveis.

  5. Node.js

    Ambiente de execução JavaScript no servidor, muito utilizado no desenvolvimento backend.

Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar conteúdos em páginas web.
CSS
Linguagem utilizada para estilizar elementos HTML.
JavaScript
Linguagem de programação voltada para interatividade em páginas web.
Também pode ser utilizada no backend com Node.js.
DOM
Estrutura em árvore que representa os elementos de uma página HTML.
API
Interface utilizada para comunicação entre sistemas.
Pode disponibilizar serviços, dados e autenticação.
HTTP
Hypertext Transfer Protocol
Protocolo responsável pela transferência de dados na web.
URL
Endereço Web
Localização de um recurso disponível na internet.
DNS
Sistema responsável por converter nomes de domínio em endereços IP.
